# Micro Sentiment App

A simple web app for analysing text sentiment in real time.

Users can input text and get a sentiment classification (positive, negative, or neutral), with a lightweight frontend and a basic processing pipeline behind it.

## Features

- Real-time sentiment analysis  
- Clean and minimal UI  
- Fast response for short text inputs  

## Tech

- React  
- Tailwind CSS  
- Python (backend/processing)  

## Run locally

```bash
npm install
npm start
